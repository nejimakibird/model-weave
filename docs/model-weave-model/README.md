## Summary

**重要:**

*   このフォルダ内のファイルは、Model Weave の正式な仕様ではありません。
*   Model Weave の正式な仕様は、`docs/formats/` ディレクトリ内のファイルおよびリポジトリのルートにある `README.md` です。
