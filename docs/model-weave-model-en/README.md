# Model Weave Model

This folder organizes the specification and implementation structure of the Obsidian plugin Model Weave as Model Weave Markdown model files. It is intended for dogfooding Model Weave and for validating reverse-engineering use cases.

## Summary

**Important:**

*   The files in this folder are not the formal Model Weave specification.
*   The formal Model Weave specification is defined by the files in `docs/formats/` and the root `README.md` of the repository.
*   This model acts as supporting documentation for understanding Model Weave internals and informing future design and improvements.
